Documentation

Welcome to the ModernOS documentation. ModernOS is a next-generation operating system built from scratch with a custom kernel and a Flutter-based user interface.

Installation

Currently, ModernOS is distributed as a live ISO image. You can download the latest release from our GitHub repository.

Running in QEMU

The easiest way to try ModernOS is using QEMU. We provide a helper script for this.

./run.sh

Or manually:

qemu-system-x86_64 -drive file=ModernOS-1.0.iso,format=raw,media=cdrom,readonly=on -m 256M -serial stdio -vga std

Kernel

Our kernel is a custom 64-bit kernel written in C++ and Assembly. It features a hybrid architecture with a focus on low latency and real-time capabilities.

Flutter Embedder

The UI is powered by a custom Flutter embedder that runs directly on the kernel's framebuffer. This allows for a smooth, hardware-accelerated experience without the overhead of a traditional windowing system like X11 or Wayland.
